# The Coding Ledger

A modern tech blog platform built with Next.js and Express.js featuring authentication, articles, comments, likes, and newsletter management.

## Overview

This repository contains a full-stack blog platform with:

- **Frontend**: Next.js application with modern UI/UX
- **Backend**: Express.js API with MongoDB/Prisma
